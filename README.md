Northwest is a game created using typical web technologies, namely React and ASP.NET. I chose to use these odd technologies, for game, because all the interactions occur in the UI. Since React is built for this exact use case, it's a logical pick. The alternative would have been to use a game engine, which suit graphics rendering and audio processing better. However, these elements are not crucial for Northwest. It has to be thought of as a digital board-game where the cards are represented through a UI and the game's rules are controlled through programming.
